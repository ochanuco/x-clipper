# やりたいことメモ

## 完了
- Chrome 拡張機能で X 投稿詳細ページから必要情報を抽出し、Notion に保存する処理を実装。
- 抽出対象: スクリーンネーム、ユーザーネーム、本文、投稿時刻、画像（最大 4 枚）。
- Notion には本文・画像をページ本文、その他はプロパティとして書き込み。
- 右クリックメニューと拡張機能ボタンの両方から実行できるフローを構築。
- 個人利用向けに開発者モードで読み込む運用を想定。
- Notion の DB URL を保存し ID を抽出するオプション UI を実装。
- TypeScript + pnpm での開発環境、ビルドスクリプトを整備。

## 次に着手したいこと
- Notion のインテグレーションキーと対象データベースの連携設定（Notion 側での共有を含む）。
- `dist/` を Chrome のデベロッパーモードで読み込み、実環境でクリップ動作を確認。
- 自動テスト基盤（Jest + MSW）と最低限のユニットテストを追加。

## 後で検討すること
- ウォッチビルドやホットリロードなど開発体験向上のためのツール導入。
- UI/UX 向上のための通知改善やクリップ完了後の Notion ページリンク表示。
